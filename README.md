# 📊 Data Professional Survey Analysis Dashboard

## 📌 Project Overview
This project provides a comprehensive analysis of a global survey conducted among data professionals (Data Analysts, Data Scientists, Engineers, etc.). The Power BI dashboard visualizes key trends regarding salaries, job satisfaction, programming preferences, and demographic distributions to provide a clear picture of the current data industry landscape.

## 🎯 Objectives
* **Analyze Salary Trends:** Understand how compensation varies across different roles and countries.
* **Evaluate Job Satisfaction:** Measure happiness levels regarding salary, work-life balance, and career growth.
* **Identify Tool Preferences:** Determine the most popular programming languages among professionals.
* **Break Down Demographics:** Explore the educational background, age, and gender distribution within the data field.
* **Career Entry Analysis:** Understand the perceived difficulty of breaking into the data industry.

## 📁 Dataset
The dataset originates from a "Data Professional Survey" and includes the following key information:
* **Demographics:** Age, Gender, Ethnicity, Country, and Education.
* **Professional Details:** Current Job Title, Industry, and Career Switch status.
* **Financials:** Yearly Salary (binned ranges in USD).
* **Sentiment:** Happiness ratings (1-10) for Salary, Work/Life Balance, Management, and Upward Mobility.
* **Technical:** Favorite Programming Language and difficulty of breaking into the field.

## 📊 Dashboard Features
* **KPI Cards:** Total survey participants, average age, and satisfaction scores.
* **Salary by Job Title:** A bar chart comparing earnings across different data roles.
* **Programming Language Popularity:** A visual representation of preferred tools (e.g., Python, R, SQL).
* **Global Distribution:** A map view showing where survey respondents are located.
* **Happiness Score Radar/Gauge:** Average satisfaction levels across different workplace categories.
* **Education Level Breakdown:** A donut chart showing the highest level of education attained.

## 🛠️ Tools Used
* **Power BI:** Data modeling, DAX calculations, and interactive visualization.
* **Excel / CSV:** Data source and initial inspection.

## 📌 Key Insights (Sample)
* **Python Dominance:** Python remains the most preferred programming language among data professionals.
* **Salary Gaps:** Significant variations in salary are observed based on geographical location (e.g., US vs. India) and job title.
* **Work-Life Balance:** Generally, respondents score work-life balance higher than upward mobility or salary satisfaction.
* **Education:** A high percentage of data professionals hold a Bachelor’s or Master’s degree, but many are self-taught or career-switchers.

## 📂 Files in Repository
* **9.pbix:** The main Power BI dashboard file.
* **9. Power BI - Final Project.xlsx - Data Professional Survey.csv:** The raw survey data used for analysis.

## 🚀 How to Use
1.  **Download** the `.pbix` file.
2.  **Open** the file using **Power BI Desktop**.
3.  **Interact** with the filters (Slicers) on the side or top to filter data by Country, Job Title, or Education level.
